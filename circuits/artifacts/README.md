# Circuit artifacts — DEVELOPMENT KEYS

Generated by `scripts/build-artifacts.sh` from a fixed public
seed: reproducible by anyone, secure for no one. The setup
randomness is public, so **these keys must never gate real
value**. Production keys come from the multi-party ceremony
described in `../ceremony/`.

Per circuit: `verifying_key.hex` (host-encoded fields, what a
`zk_verifier` instance is constructed with), `verifying_key.bin`
(arkworks canonical), `proving_key.bin` (arkworks canonical,
gitignored for size — regenerate with the script), `layout.txt`
(public-input order), and `SHA256SUMS`.
